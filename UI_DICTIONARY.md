# Wordle Duel UI Dictionary

Use these terms when describing changes or bugs.

## Main Menu

- **Main menu**: The first screen with the Wordle Duel title and menu panel.
- **Name gate**: First menu step where the player enters or skips their display name.
- **Brand block**: Left-side title area with "Wordle Duel", headline, and description.
- **Menu panel**: Right-side control panel with theme, player name, and menu buttons.
- **Menu step**: One state inside the menu panel, such as name entry, main options, or duel options.
- **Theme toggle**: Dark/light mode switch above the player name field.
- **Player name field**: Input where the player enters their display name.
- **Continue button**: Saves the entered player name and opens the main menu.
- **Skip Name button**: Uses the default player name and opens the main menu.
- **Daily Challenge button**: Starts the solo daily puzzle.
- **2 Player Duel button**: Opens the duel submenu.
- **Duel submenu**: Menu step with Create Room and Join Room options.
- **Create Room button**: Creates a multiplayer room.
- **Join Room button**: Reveals the room-code entry flow.
- **Duel Back button**: Returns from the duel submenu to the main menu.
- **Room code field**: Input where a player enters a room code.
- **Join button**: Green button that joins a room using the code field.
- **Leaderboard button**: Opens the leaderboard screen.
- **Menu status text**: Small feedback text below the menu controls.

## Game Header

- **Game header**: Top area inside the game screen.
- **Back button**: Arrow button that returns to the main menu.
- **Mode label**: Small label above the title, such as "Solo" or "Room".
- **Game title**: Main title in the game screen.
- **Room badge**: Room-code pill next to "Wordle Duel" in multiplayer.
- **Room code display**: The visible multiplayer room code inside the room badge.
- **Copy room button**: Button inside the room badge that copies the code.
- **Timer**: Solve timer shown in the header.

## Game Board

- **Message text**: Feedback line above the board.
- **Main board**: The 6-by-5 Wordle grid.
- **Attempt row**: One horizontal row on the main board.
- **Letter tile**: One square in the main board.
- **Empty tile**: A tile before a letter is typed.
- **Filled tile**: A tile with a typed letter before submission.
- **Correct tile**: Green tile.
- **Present tile**: Yellow/gold tile.
- **Absent tile**: Dark grey tile.

## Keyboard

- **Keyboard**: On-screen letter keyboard.
- **Key row**: One horizontal row of keyboard keys.
- **Letter key**: A normal alphabet key.
- **Enter key**: Submits the current guess.
- **Backspace key**: Deletes the last typed letter.
- **Unused key**: Key that has not appeared in a submitted guess yet.
- **Used absent key**: Dark grey key for a letter confirmed absent.
- **Correct key**: Green key for a confirmed correct-position letter.
- **Present key**: Yellow/gold key for a confirmed present letter.

## Opponent Panel

- **Opponent panel**: Multiplayer progress box.
- **Opponent name**: Name/status text inside the opponent panel.
- **Opponent progress grid**: Mini Wordle grid showing only colors, never letters.
- **Opponent attempt row**: One mini row for an opponent guess.
- **Opponent mini cell**: One small color dot/square in the opponent progress grid.
- **Attempt divider**: Faint line separating opponent attempts.

## Leaderboard

- **Leaderboard screen**: Page showing player rankings.
- **Leaderboard table**: The ranked list area.
- **Leaderboard header row**: Column labels.
- **Leaderboard player row**: One player’s aggregated score row.
- **Rank column**: Player position.
- **Name column**: Player display name.
- **Total points column**: Accumulated points for that name.
- **Games column**: Number of solved games for that name.
- **Best time column**: Fastest solve time.
- **Best tries column**: Lowest attempt count.

## Layout Terms

- **Desktop layout**: Wide-screen layout.
- **Mobile layout**: Phone layout.
- **Duel room layout**: Multiplayer game screen.
- **Solo layout**: Daily challenge game screen.
- **Compact spacing**: Tighter vertical or horizontal spacing.
- **Responsive breakpoint**: Screen width where layout rules change.
